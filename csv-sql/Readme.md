# CSV SQL like engine
**CSV SQL like engine** is a mini library that has the purpose of replicating the idea of SQL engine, but done upon a CSV files.

---

## Purpose
This library is developed to experiment with **objects that manage resources** and **concurrent programming** in C++.

---

## Basic Requirements
 - Use objects to manage resources
 - use threads
