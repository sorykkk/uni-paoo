# Document Analytics
**Document Analytics** is a mini library that has the purpose of performing document analysis algorithms.

---

## Purpose
This library is developed to experiment with **objects that manage resources** and **concurrent programming** in C++.

---

## Basic Requirements
 - Use objects to manage resources (unique_ptr, shared_ptr)
 - use threads (and their sync methods: mutex)

---

## Features
 - Performs tf-idf analysis
---
