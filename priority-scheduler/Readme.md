# Priority Scheduler Simulator
**Priority Scheduler Simulator** is a mini library that has the purpose of simulating a priority scheduler manager as an object that can be implemented in a system without one (if there are one, which has thread capability and no scheduler).

---

## Purpose
This library is developed to experiment with **objects that manage resources** and **concurrent programming** in C++.

---

## Basic Requirements
 - Use objects to manage resources
 - use threads
